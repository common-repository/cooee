== Description ==

Let's Cooee enables brands to deliver One-To-One personalisation powered by AI. With Canva like design interface that
combines with data to generate engagements for each user across digital channels, uniquely within brining in automation.

This plugin uses Letscooee’s Web SDK. Let's Cooee is an AI-based engagement platform that uses multiple ML algorithms
to create personalised engagement for each customer. It uses Augmented Reality, In-App and Push Notification as the
medium of engagement. In order to make personalised engagement, Cooee gathers multiple data of the customer and creates
a customer journey, predicts its next move, and figures out the best time to engage customers. The medium of engagement
is also AI-driven.
